# 🧠 Auth0 AI Agent Assistant

This project is a submission for the **Auth0 for AI Agents Challenge**, showcasing a secure, user-controlled AI assistant powered by the Gemini API and Auth0 authentication.

It demonstrates how to build trustworthy agentic systems with:

* 🔐 Granular permission scoping via Auth0
* 🛠 Dynamic tool and knowledge source management
* 🧭 Transparent boundaries enforced through system instructions
